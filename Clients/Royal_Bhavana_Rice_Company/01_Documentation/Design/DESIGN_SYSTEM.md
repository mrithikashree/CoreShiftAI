\# Royal Bhavana Rice Company

\# Design System



\---



\# Purpose



This document defines the visual design standards for the Royal Bhavana Rice Company Digital Sales \& Lead Management Platform.



The objective is to maintain consistency across the website, CRM, reports, and future applications.



\---



\# Brand Personality



The interface should communicate:



\- Trust

\- Quality

\- Simplicity

\- Professionalism

\- Freshness

\- Reliability



\---



\# Color Palette



\## Primary



Green



Purpose

\---



\## Secondary

\---



\## Neutral



White



Light Gray



Dark Gray

Backgrounds and readable text.

\# Typography



\## Heading



Bold



Easy to read



Professional



\---



\## Body



Clean



Simple



Readable



\---



\# Button Styles



\## Primary Button



\- Filled

\- Rounded corners

\- Strong call-to-action



Examples



\- Buy Now

\- Enquire Now



\---



\## Secondary Button



Outlined



Examples



\- Learn More

\- View Products



\---



\# Form Controls



Text Boxes



Dropdowns



Text Areas



Checkboxes



Radio Buttons



All controls should:



\- Have consistent height

\- Have rounded corners

\- Show validation messages clearly



\---



\# Cards



Used for:



\- Products

\- Testimonials

\- Marketplace Links

\- Benefits



Card Rules



\- Rounded corners

\- Soft shadow

\- Equal spacing

\- Responsive



\---



\# Icons



Simple



Modern



Consistent



Outline style preferred.



\---



\# Images



Use



\- High-quality product photos

\- Lifestyle imagery

\- Optimized file sizes



Avoid blurry or stretched images.



\---



\# Spacing



Maintain consistent spacing throughout the application.



Avoid cluttered layouts.



\---



\# Border Radius



Use a consistent rounded style across:



\- Buttons

\- Cards

\- Input fields

\- Images



\---



\# Shadows



Apply subtle shadows only where needed.



Avoid heavy shadow effects.



\---



\# Responsive Design



Desktop



Tablet



Mobile



Design mobile-first.



\---



\# Accessibility



\- Good contrast

\- Readable font sizes

\- Large touch targets

\- Keyboard-friendly (future)



\---



\# Design Principles



Every screen should be:



\- Clean

\- Simple

\- Consistent

\- Fast

\- Easy to understand



\---



\# UI Consistency Rules



Every page must use:



\- Same header

\- Same footer

\- Same button styles

\- Same spacing

\- Same typography

\- Same color palette

\- Same icon style



No page should introduce new visual styles without updating this document.



\---



\# Success Criteria



A user should immediately:



\- Recognize the brand

\- Navigate confidently

\- Read content comfortably

\- Take action easily
