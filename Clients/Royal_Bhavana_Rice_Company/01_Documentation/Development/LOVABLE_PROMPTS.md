\# Royal Bhavana Rice Company

\# Lovable Prompt Library



\---



\# Purpose



This document contains all Lovable prompts used to build the Royal Bhavana Rice Company Digital Sales \& Lead Management Platform.



The prompts are organized feature-by-feature to support consistent development and future reuse.



\---



\# Prompt Writing Standards



Every prompt should include:



\- Objective

\- Design Requirements

\- Functional Requirements

\- Mobile Responsiveness

\- Accessibility

\- Expected Output



\---



\# Prompt 001 – Project Initialization



\## Objective



Create the initial project structure.



\## Prompt



Create a responsive landing website for Royal Bhavana Rice Company.



The website should represent a premium rice brand with a clean, modern and trustworthy appearance.



Use a mobile-first responsive layout.



Prepare reusable components that can be extended in future.



Generate semantic HTML structure and modern UI.



Do not include placeholder sections unless specified.



\---



\# Prompt 002 – Header



\## Objective



Create the website header.



\## Prompt



Design a sticky responsive navigation bar.



Include:



\- Company Logo

\- Home

\- About

\- Products

\- Testimonials

\- Contact



On the right side include:



\- WhatsApp Button



\- Buy Online Button



Navigation should collapse into a mobile hamburger menu.
